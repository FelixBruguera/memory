A Dragon Ball themed memory game

## Live demo

[Check out the live demo](https://memory-18o.pages.dev/)

## Features

- **Multiple levels**: The game has a total of 6 levels
- **Difficulty selector**: Three dificulty levels with a different time limits and score multipliers
- **Score tracking**: The game keeps track of the current and best scores
- **Timer**: Medium and Hard difficulties have a time limit between plays
